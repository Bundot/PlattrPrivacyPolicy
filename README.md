# Plattr Privacy Policy — Local Files

This directory contains a standalone HTML privacy policy for Plattr Technologies Ltd.

Files created:
- `privacy-policy.html` — The privacy policy content. Open this file in any browser to view.
- `styles/privacy-policy.css` — Styles used by the HTML file.

Quick start

Open the HTML file in your browser:

```bash
open privacy-policy.html
```

Editing

- Update the placeholders in `privacy-policy.html` for Effective Date, Last Updated, Email, and Address.
- For minor style tweaks, edit `styles/privacy-policy.css`.

If you want this integrated into a website, copy the HTML into your site's template and place the CSS into your global styles or import it from `/styles/privacy-policy.css`.
